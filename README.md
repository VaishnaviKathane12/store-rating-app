# Store Rating App 🛒⭐

A fullstack application for rating and reviewing stores.

## 🚀 Tech Stack
- **Backend:** Node.js, Express.js, PostgreSQL
- **Frontend:** React.js, Context API
- **Database:** PostgreSQL
- **Auth:** JWT Authentication

---

## 📂 Project Structure

├── frontend/ # React.js frontend
├── migrations/ # DB migration files
